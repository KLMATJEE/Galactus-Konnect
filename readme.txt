Galactus Konnect Responsive HTML5 Template
Galactus Konnect html5 website template is a multi-purpose clean html5 theme built with valid HTML5 & CSS3. 
This design has a tech industry modern look, full-fill with latest standards.
It's based on personal project to illustrate skills in UI/UX Design (Bootstrap framework) fully responsive web compatible with multi browser and devices.
This website template uses HTML5 ,Bootstrap CSS3, JQuery 1.8.1 version & Javascript to illustrate mobile website and start-up company website front end.

Key features
-------------
Google Maps API
Clean & Developer-friendly HTML5 and CSS3 code
100% Responsive Layout Design 
Multi-purpose Social media linkback
Google Fonts Support
Font Awesome 
Smooth Scrolling 
Fully Customizable
Contact Form
